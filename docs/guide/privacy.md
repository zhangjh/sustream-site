# 隐私安全说明

PureFlow (素流) 秉承“隐私即权利”的理念，致力于为您提供一个完全掌控个人数字资产的纯净环境。

## 1. 核心承诺

- **零广告，零追踪**：应用内没有任何形式的广告、数据追踪 SDK 或用户行为分析。
- **数据本地化**：您的所有文件数据、操作记录、偏好设置均存储在设备本地，绝不上传至任何服务器。
- **无隐私窃取**：基础版应用在默认情况下无需任何网络权限，从设计层面杜绝了数据泄露的可能性。

## 2. 权限使用

我们严格控制应用所需权限，仅在必要时请求，并明确告知其用途。

- **文件存储权限**：这是 PureFlow 作为文件管理器的核心功能所需。用于：
  - 浏览、创建、修改、删除、移动和复制您的本地文件及文件夹。
  - 扫描存储空间以进行深度分析（如大文件、重复文件识别）。
- **网络权限 (Pro 版 AI 引擎)**：仅当您在 Pro 版中主动启用 AI 引擎，并配置了您自己的大模型 API Key 后，应用才会进行网络连接。
  - 传输内容仅为结构化的 Agent 指令与文件路径元数据（例如：“将 `Download/invoice.pdf` 移动到 `Documents/Finance`”）。
  - **您的文件内容（图片、文档文本等）绝不会未经您的明确同意直接上传至云端。**

## 3. 端侧 AI 与数据隐私

Pro 版的端侧 AI 语义检索引擎（基于本地向量库）在设备本地完成文件的特征提取和向量化。这意味着：

- **隐私安全**：您的文件内容，甚至用于生成向量的中间数据，都不会离开您的设备。AI 的“思考”过程，完全在本机完成。
- **离线能力**：即使在没有网络连接的情况下，本地语义检索功能也能正常工作。

## 4. 开源与透明 (未来计划)

PureFlow (素流) 未来计划逐步开源核心代码，接受社区审计，确保技术栈的透明与安全。

**您的信任，是我们的生命线。**
