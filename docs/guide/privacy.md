# 隐私安全说明

素流 (PureFlow) 秉承"隐私即权利"的理念，致力于为您提供一个完全掌控个人数字资产的纯净环境。本页面详细说明我们的隐私承诺、权限使用方式和数据处理机制。

## 核心承诺

### 零广告零追踪

素流 (PureFlow) 不包含任何形式的广告 SDK，不集成任何用户行为分析组件。应用内没有广告展示、没有埋点追踪、没有数据采集——您的每一次操作都不会被记录和分析。

### 数据本地化

您的所有数据均存储在设备本地：

- **文件数据**：您管理的所有文件始终保留在设备存储中
- **操作记录**：文件操作历史、搜索记录等仅存储在本地
- **偏好设置**：应用配置、界面偏好等设置数据保存在设备本地

没有任何数据会被自动上传到云端服务器。

### 无隐私窃取

素流 (PureFlow) 从设计层面杜绝隐私泄露的可能性。所有功能均在本地运行，不存在后台偷传数据的行为。我们不收集、不存储、不分析您的任何个人信息。

## 权限说明

素流 (PureFlow) 仅申请运行所必需的权限，以下是应用声明的全部权限及其用途：

### MANAGE_EXTERNAL_STORAGE — 文件管理核心权限

作为文件管理器，素流 (PureFlow) 需要访问设备上的所有文件，包括内部存储和外部存储中的文件。这是文件浏览、复制、移动、删除等核心功能的基础权限。

### QUERY_ALL_PACKAGES — 获取应用图标显示目录

Android 11 及以上版本引入了包可见性限制。素流 (PureFlow) 需要此权限来查询已安装应用的信息，以便在浏览 APK 文件或应用相关目录时正确显示对应的应用图标。

### INTERNET — 网络权限

素流 (PureFlow) 的网络权限用于以下三个场景：

- **付费激活码获取**：Pro 版购买激活时需要联网验证激活码
- **Pro 版 BYOK 模式**：当用户配置了自己的 LLM API Key 后，调用用户自配的 LLM API 端点（数据直接发送到用户配置的服务，不经过素流服务器）
- **端侧 AI 模型首次下载**：端侧 AI 功能所需的模型文件在首次使用时从服务器下载到本地，后续完全离线运行

::: info 说明
基础版在日常使用中无需联网，所有文件操作均在本地完成。
:::

### FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC — 大文件传输后台运行

当执行大文件的复制、移动等耗时操作时，素流 (PureFlow) 需要前台服务权限来保持后台运行，防止系统在操作过程中终止应用，确保文件传输的完整性。

### POST_NOTIFICATIONS — 通知权限

Android 13 及以上版本需要显式申请通知权限。素流 (PureFlow) 使用通知来告知您文件操作的进度和完成状态，例如大文件复制、后台索引构建等任务的进展。

## 端侧 AI 隐私保障

素流 (PureFlow) Pro 版的所有 AI 功能（语义搜索、相似图片检测、智能清理建议等）均采用端侧 AI 技术，在您的设备本地完成特征提取和向量化处理。这意味着：

- **文件内容不出端**：您的图片、文档等文件内容仅在设备本地被 AI 模型处理，生成的特征向量也存储在本地，任何数据都不会离开您的设备
- **离线可用**：模型下载到本地后，语义搜索、相似图片检测等 AI 功能无需网络连接即可正常使用
- **中间数据本地留存**：AI 处理过程中产生的所有中间数据（特征向量、索引文件等）均保存在设备本地，不会上传到任何服务器

## BYOK 模式数据流向

素流 (PureFlow) Pro 版的 BYOK（Bring Your Own Key）模式允许您使用自己的 LLM API Key。在此模式下：

1. **您自行配置 API Key**：在应用内填入您自己的 LLM API Key 和 API 端点地址
2. **数据直达您的 API**：对话请求直接从您的设备发送到您配置的 API 端点
3. **不经过素流服务器**：整个通信过程中，素流 (PureFlow) 的应用服务器不参与任何数据中转，您的数据完全在您和您选择的 LLM 服务之间流转

## 模型下载数据流向

端侧 AI 功能所需的模型文件（如 MobileCLIP2 语义检索模型、BGE 文档检索模型等）采用以下数据流向：

- **下载方向**：模型文件从服务器下载到您的设备本地
- **不涉及上传**：下载过程中不会上传任何用户数据，服务器仅提供模型文件的下载服务
- **一次下载，永久离线**：模型下载完成后保存在设备本地，后续使用完全离线，无需再次联网

## 本地运行与操作安全

素流 (PureFlow) 的所有功能均在本地运行，不存在数据泄露风险：

- **文件操作本地执行**：浏览、复制、移动、删除、压缩等所有文件操作均在设备本地完成
- **AI 推理本地运行**：语义搜索、相似图片检测、智能聚合等 AI 功能在设备端完成推理
- **破坏性操作二次确认**：所有可能造成数据损失的操作（如删除文件、批量移动等）均需要您的明确确认后才会执行，防止误操作
