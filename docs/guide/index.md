# 什么是素流 (PureFlow)？

素流 (PureFlow) 是一款面向 Android 平台的 **AI 原生文件资产管家**。它将端侧 AI 推理与云端大模型深度融合，在保障隐私安全的前提下，为你提供从基础文件管理到智能语义检索、对话式文件操作的全方位能力。

## 核心架构理念

**"大脑指挥肌肉"** —— 这是素流 (PureFlow) 的核心设计哲学。

- **云端大模型（大脑）**：负责理解用户的自然语言意图，进行推理与任务规划，将复杂需求拆解为可执行的操作序列。
- **端侧原生代码（肌肉）**：负责安全、高效地执行文件操作。所有文件访问、数据处理均在本地完成，确保用户数据不出端。

这种端云协作架构让素流 (PureFlow) 既拥有大模型的智能理解能力，又保持了原生应用的安全性与响应速度。

## 基础版核心能力

素流 (PureFlow) 基础版提供完整的对标ES文件浏览器的本地文件管理体验：

- **本地文件管理**：支持多标签页浏览、跨目录剪贴板，让文件操作像浏览器一样灵活高效。
- **格式直开**：内置 ePub 阅读器、PDF 查看器、压缩包免解压预览、文本/图片/音视频播放，无需跳转第三方应用。
- **深度存储分析与清理**：可视化存储占用分布，智能识别大文件、重复文件、缓存垃圾，帮你释放宝贵空间。

## Pro 版 AI 能力

升级 Pro 版，解锁端侧 AI 与云端智能的全部潜力：

- **端侧语义检索**：基于 MobileCLIP2 模型在本地完成图片/视频的向量化，支持自然语言搜索，中英文通用，离线可用。
- **相似图片检测与智能清理**：自动识别近重复和语义相似的图片，智能推荐保留策略，告别手动逐张筛选。
- **素流助理 AI Agent**：对话式文件管理，用自然语言描述需求即可完成批量操作，所有破坏性操作需用户确认，安全可控。
- **智能伴读**：为 ePub 书籍生成内容总结，支持基于书籍内容的智能问答，让阅读更高效。
- **素匣隐私保险箱**：AI 驱动的隐私文件扫描（身份证、银行卡、证件等），配合密码保护的加密存储，守护你的敏感信息。
- **文档语义搜索**：基于 BGE 模型的文档内容检索，支持中英翻译，按语义而非关键词匹配文档内容。

## 技术栈亮点

- **Kotlin + Jetpack Compose**：100% 原生开发，Material Design 3 设计语言，流畅的交互体验与深色模式支持。
- **VFS 统一文件系统抽象层**：屏蔽 Android 碎片化的存储访问差异，为上层功能提供一致的文件操作接口。
- **ONNX Runtime Mobile + MobileCLIP2 端侧推理**：AI 模型在设备本地运行，文件内容不出端，兼顾智能与隐私。
- **BYOK 零加价模式**：用户自带 API Key 调用云端大模型，数据直接发送到用户配置的 API 端点，不经过素流 (PureFlow) 服务器，零中间商加价。
