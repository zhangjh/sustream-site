## 权限说明

素流 (PureFlow) 仅申请运行所必需的权限，不包含任何广告 SDK 或用户行为分析组件。以下是应用声明的全部权限及其用途：

### MANAGE_EXTERNAL_STORAGE — 文件管理核心权限

作为文件管理器，素流 (PureFlow) 需要访问设备上的所有文件，包括内部存储和外部存储中的文件。这是文件浏览、复制、移动、删除等核心功能的基础权限。

### QUERY_ALL_PACKAGES — 获取应用图标显示目录

Android 11 及以上版本引入了包可见性限制。素流 (PureFlow) 需要此权限来查询已安装应用的信息，以便在浏览 APK 文件或应用相关目录时正确显示对应的应用图标。

### INTERNET — 网络权限

素流 (PureFlow) 的网络权限用于以下三个场景：

- **付费激活码获取**：Pro 版购买激活时需要联网验证激活码
- **Pro 版 BYOK 模式**：当用户配置了自己的 LLM API Key 后，调用用户自配的 LLM API 端点（数据直接发送到用户配置的服务，不经过素流服务器）
- **端侧 AI 模型首次下载**：端侧 AI 功能所需的模型文件在首次使用时从服务器下载到本地，后续完全离线运行

::: info 说明
基础版在日常使用中无需联网，所有文件操作均在本地完成。
:::

### FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC — 大文件传输后台运行

当执行大文件的复制、移动等耗时操作时，素流 (PureFlow) 需要前台服务权限来保持后台运行，防止系统在操作过程中终止应用，确保文件传输的完整性。

### POST_NOTIFICATIONS — 通知权限

Android 13 及以上版本需要显式申请通知权限。素流 (PureFlow) 使用通知来告知您文件操作的进度和完成状态，例如大文件复制、后台索引构建等任务的进展。

## 本地 APK 安装说明

从国内加速下载渠道获取的是 `.apk` 安装包文件，部分国产手机系统会对非应用商店来源的安装包进行拦截。以下是安装步骤：

### 1. 允许安装未知来源应用

首次安装时，系统会提示「不允许安装未知来源的应用」。请按照提示前往设置，为您使用的浏览器或文件管理器开启「允许安装未知应用」权限。

不同品牌的路径略有差异，通常在：**设置 → 安全 → 更多安全设置 → 安装未知应用**

### 2. 处理厂商安全拦截

部分国产手机厂商（如华为、OPPO、vivo、小米等）会在安装时弹出风险提示或直接阻止安装。常见处理方式：

| 品牌 | 处理方式 |
|------|----------|
| 华为/荣耀 | 弹出「安全威胁」提示时，点击「仍然安装」 |
| 小米/红米 | 弹出「安全扫描」后等待扫描完成，选择「继续安装」；如提示需要登录小米账号验证，可在设置中关闭「安全扫描」 |
| OPPO/realme | 弹出风险提示时输入锁屏密码确认安装 |
| vivo/iQOO | 弹出风险提示时点击「继续安装」，可能需要输入锁屏密码 |

::: tip 提示
这些拦截是厂商对所有非自家应用商店来源的安装包的通用策略，并非针对素流。素流不含任何恶意代码，您可以放心安装。
:::

### 3. 安装完成

确认安装后等待安装完成即可打开使用。后续版本更新时，直接下载新版本覆盖安装，应用数据会保留。

## 常见问题

### 如何使用 ePub 阅读器？

在文件浏览器中点击任意 `.epub` 文件，素流 (PureFlow) 会自动打开内置的 ePub 阅读器。阅读器支持目录导航、记住进度等基本阅读功能。如果您已开启 Pro 版，还可以使用智能伴读功能对书籍内容进行总结和问答。

### 如何预览压缩包内容？

素流 (PureFlow) 支持 `.zip`、`.rar`、`.7z` 等常见压缩格式。在文件浏览器中点击压缩文件即可直接查看内部文件列表和目录结构，无需先解压。您也可以选择将压缩包解压到指定目录。
