# 核心应用场景

素流 (PureFlow) 的 AI 能力不是炫技，而是解决真实的文件管理痛点。以下 5 个场景展示了素流 (PureFlow) 如何将端侧 AI 和云端智能结合，在保护隐私的前提下让文件管理变得高效、智能。

## 场景一：模糊记忆的语义检索

### 痛点

几周前保存了一张朋友发来的风景照，或者截了一张产品设计稿，但完全想不起文件名叫什么。传统文件搜索只能按文件名匹配——你输入"风景"，它只会找名字里带"风景"两个字的文件，而那张照片可能叫 `IMG_20250615_143022.jpg`。

更常见的情况是：你记得图片的内容（"一只趴在键盘上的橘猫"、"上次团建的合照"），却无法用文件名表达这些记忆。面对几千张图片，只能一页一页翻找，效率极低。

### 素流 (PureFlow) 的解决方式

打开素流 (PureFlow) 的语义搜索，直接输入你脑海中的描述：

> "海边的日落"
> "一只橘猫"
> "会议室白板上的流程图"

系统会理解你描述的语义含义，从图库中找出内容最匹配的图片和视频，即使文件名是毫无意义的 `IMG_xxxx.jpg`。支持中英文混合查询，"sunset at beach"和"海边日落"能找到同一张照片。

### 技术原理

语义检索基于端侧 MobileCLIP2 模型实现，整个过程完全在设备本地完成：

1. **端侧向量化**：MobileCLIP2 模型将每张图片编码为高维语义向量，捕捉图片的视觉内容特征。模型运行在设备本地，基于 ONNX Runtime Mobile 推理引擎，单张图片推理延迟约 50-100ms
2. **本地语义索引**：向量化结果存储在本地数据库中，支持增量更新，新增图片自动纳入索引
3. **自然语言匹配**：输入的文字描述同样被编码为语义向量，通过余弦相似度计算找出语义最接近的结果

**隐私保障**：图片内容的向量化在端侧完成，原始图片和语义向量均不上传到任何服务器。整个检索过程离线可用，无需网络连接。

## 场景二：批量文件整理

### 痛点

手机下载目录里堆满了各种文件：发票截图、工作文档、安装包、社交应用保存的图片……手动整理意味着：长按选择文件 → 新建文件夹 → 重命名 → 拖拽移动，每个文件都要重复这套操作。如果要按规则批量重命名（比如"日期_类型_序号"），更是一场噩梦。

传统的自动化工具要么需要编写脚本，要么只支持简单的按扩展名分类，无法理解文件的实际内容和用途。

### 素流 (PureFlow) 的解决方式

唤醒素流助理 AI Agent，用自然语言描述你的整理需求：

> "把下载目录里所有和报销相关的发票截图和 PDF，以'日期_报销凭证'的格式重命名，然后统一移动到 Finance 文件夹下。"

> "找出最近一个月拍的所有美食照片，按餐厅名称分类整理到 Food 文件夹的子目录里。"

> "把 Documents 里超过半年没打开的 PDF 文件归档到 Archive 目录。"

Agent 会理解你的意图，自动规划执行步骤，调用端侧工具完成筛选、重命名、移动等一系列操作。整个过程你只需要在最终确认环节点一下"执行"。

### 技术原理

批量文件整理依赖素流助理 AI Agent 的端云协作架构：

1. **云端意图理解**：大语言模型（通过 BYOK 模式调用）接收你的自然语言指令，理解"筛选条件 + 命名规则 + 目标位置"等复合意图，将其拆解为结构化的工具调用计划
2. **端侧智能筛选**：Agent 调用端侧语义搜索工具定位目标文件（如识别"发票"特征的图片），结合文件搜索工具按时间、大小、类型等条件精确过滤
3. **端侧安全执行**：筛选和操作计划确定后，由端侧原生代码调用 Android API 执行文件操作。所有破坏性操作（移动、删除、重命名）需要你手动确认后才会执行

**隐私保障**：云端大模型只接触经过脱敏处理的结构化元数据（文件类型、大小区间、时间区间、目录标签等），不接触原始文件内容、完整文件路径和缩略图。实际的文件访问和操作全部在端侧完成。

## 场景三：智能存储清理

### 痛点

手机提示存储空间不足，打开系统自带的清理工具，看到的是一堆"缓存文件 2.3GB"、"其他文件 5.1GB"这样的笼统分类。你不知道这些文件具体是什么，不敢贸然删除，生怕误删重要数据。

更隐蔽的空间杀手是图库里的冗余图片：连拍的 20 张几乎一样的照片只需要保留 1 张，社交应用自动保存的图片和相机原图重复了，截图转发了好几次产生了多份副本……这些"隐形垃圾"手动排查几乎不可能。

### 素流 (PureFlow) 的解决方式

素流 (PureFlow) 提供两层智能清理能力：

**相似图片检测**：自动扫描图库，找出精确重复、近重复（同一图片的不同压缩/裁剪版本）和语义相似（同一场景不同角度）的图片组。每组智能推荐最值得保留的那一张——优先保留高分辨率、高清晰度、来自相机目录的原图。

**智能清理建议**：系统分析存储空间，生成多类型的清理建议，每条建议都附带完整的决策依据：

| 建议类型 | 说明 | 风险等级 |
|----------|------|----------|
| 近重复图片 | 向量聚类识别的重复图片组 | 中 |
| 截图堆积 | 同一时间段大量截图的累积 | 中 |
| 社交重复 | 社交应用缓存与相机原图的重复副本 | 低 |
| 连拍冗余 | 短时间连续拍摄的高度相似照片 | 中 |
| 模糊低质量 | 极度模糊或分辨率过低的图片 | 低 |
| 缓存与临时文件 | 应用缓存、缩略图、临时文件、旧安装包 | 低 |

每条建议都标注了原因说明、预计可释放空间、置信度和风险等级，你可以逐条审核后决定是否清理。

### 技术原理

智能存储清理结合了端侧 AI 检测和规则引擎：

1. **端侧向量检测**：利用 MobileCLIP2 语义向量和感知哈希（pHash）综合判定图片相似度，将相似图片分为精确重复、近重复、语义相似三个层次
2. **多维度评分**：对每组相似图片，基于分辨率、清晰度（拉普拉斯方差）、来源目录、修改时间等维度综合评分，推荐最佳保留项
3. **规则引擎分析**：结合文件元数据（大小、时间、路径、类型）生成各类清理建议，每条建议附带可追溯的决策依据

**隐私保障**：相似图片检测和清理建议的生成全部在端侧完成，不上传任何图片内容或文件信息。所有清理操作需要你手动确认后才会执行，系统不会自动删除任何文件。

## 场景四：长文档智能伴读

### 痛点

手机里存了几百页的行业报告（PDF）或几十万字的电子书（ePub），但你只有碎片时间。想快速了解核心观点，却不得不从头翻到尾；想找某个具体结论，却不知道在哪一章；看到一半放下，下次再看已经忘了前面的内容。

传统的阅读器只提供翻页和书签功能，无法帮你"理解"内容。

### 素流 (PureFlow) 的解决方式

在素流 (PureFlow) 内置的阅读器中打开文档，使用 AI 伴读功能：

**内容总结**：
> "帮我总结这篇报告的核心观点，提炼三个关键结论。"
> "这一章讲了什么？用三句话概括。"

**智能问答**：
> "这篇报告关于'下沉市场策略'的结论是什么？"
> "作者对 AI 在教育领域的应用持什么态度？"
> "张三和李四在哪一章产生了分歧？他们各自的立场是什么？"

AI 会基于文档的实际内容回答你的问题，而不是泛泛而谈。回答会关联到文档中的具体段落，帮你快速定位原文。

### 技术原理

智能伴读采用端云协作的 RAG（检索增强生成）架构：

1. **端侧内容解析**：素流 (PureFlow) 原生支持解析 ePub 和 PDF 格式，在端侧提取干净的文本内容并进行分段切片
2. **端侧轻量检索**：根据你的问题，在端侧对文档切片进行语义匹配，找出最相关的段落
3. **云端推理生成**：将相关段落（而非整篇文档）通过 BYOK 模式发送给大语言模型，生成精准的总结或回答

**隐私保障**：文档的完整内容不会整体上传到云端。端侧先完成内容切片和相关性检索，只有与问题相关的少量段落片段会发送给你自己配置的 LLM API 端点（BYOK 模式），数据不经过素流 (PureFlow) 的服务器。

## 场景五：隐私文件保护

### 痛点

手机里难免存有身份证照片、银行卡照片、护照扫描件、合同文件等敏感资料。这些文件散落在相册、下载目录、聊天记录保存的各个角落，你甚至可能忘了自己存过哪些敏感文件。

一旦手机借给他人使用、连接公共 WiFi 传输文件、或者设备丢失，这些敏感文件就面临泄露风险。普通的文件管理器没有任何保护机制，文件对任何能访问手机的人都是透明的。

### 素流 (PureFlow) 的解决方式

素流 (PureFlow) 的素匣（隐私保险箱）提供"AI 扫描 + 加密存储"的双重保护：

**AI 隐私文件扫描**：素匣利用端侧 AI 自动扫描设备中的敏感文件，识别身份证照片（正面/反面）、银行卡照片、护照、驾照等证件照片，以及其他包含个人敏感信息的文件。扫描结果以建议形式呈现，由你决定是否将这些文件移入素匣保护。

**加密存储空间**：移入素匣的文件受独立密码保护的加密存储管理。素匣使用独立的访问密码（与设备锁屏密码分离），每次打开素匣需要验证密码。即使设备被他人访问，也无法直接查看素匣中的内容。

**随时移回原位**：移入素匣的文件也可以随时一键放回原位。

### 技术原理

素匣的隐私保护完全基于端侧能力实现：

1. **端侧 AI 识别**：利用端侧模型对图片内容进行分类识别，判断是否包含证件、银行卡等敏感信息。识别过程完全在设备本地完成，不上传任何图片内容
2. **独立访问控制**：素匣的密码验证和访问控制独立于系统层面，提供应用级别的额外保护层

**隐私保障**：素匣是素流 (PureFlow) 隐私理念的集中体现——AI 扫描在端侧完成，密码验证在端侧完成，没有任何环节涉及数据上传。你的敏感文件从识别到保护的全流程都不离开设备。
