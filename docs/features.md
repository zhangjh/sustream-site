# 功能特性

素流 (PureFlow) 提供基础版 15 项核心文件管理功能和 Pro 版 8 项 AI 智能功能，覆盖从日常文件操作到智能语义检索的完整场景。

## 基础版功能

基础版即装即用，涵盖文件管理、格式直开、存储分析等 15 项核心能力。

### 文件浏览与管理

<div style="text-align: center; margin: 1rem 0;">
  <img src="/内部存储.webp" alt="文件浏览与管理" style="max-width: 280px; border-radius: 12px; box-shadow: 0 2px 12px rgba(0,0,0,0.1);" />
</div>

支持列表视图和网格视图自由切换，面包屑导航快速定位目录层级，多标签页同时打开多个目录，轻松在不同文件夹间切换操作。

### 快速分类访问

提供收藏、图片、视频、文档、下载、垃圾箱、压缩包、安装包共 8 个快捷入口，一键直达目标分类，无需逐层翻找。

### 跨目录剪贴板

全局悬浮剪贴板，支持跨目录多次复制、一次粘贴。在不同目录间选取文件后统一粘贴到目标位置，告别反复切换目录的繁琐操作。

### 文件操作

支持复制、剪切、重命名、删除、详情查看、分享、压缩等常用操作，并支持操作撤销，误操作也能轻松恢复。

### 高级排序

按名称、大小、时间、类型四种维度排序，支持正序和倒序切换，快速定位目标文件。

### 压缩包管理

支持 zip、rar、7z 等主流格式免解压直接查看内容，一键压缩文件或文件夹，解压到指定目录，无需依赖第三方压缩工具。

### ePub 阅读器

<div style="text-align: center; margin: 1rem 0;">
  <img src="/阅读器.webp" alt="ePub 阅读器" style="max-width: 280px; border-radius: 12px; box-shadow: 0 2px 12px rgba(0,0,0,0.1);" />
</div>

原生 ePub 阅读器，沉浸式阅读体验，无需安装额外阅读应用即可打开和阅读电子书。

### PDF 查看器

内置 PDF 查看器，快速打开和浏览 PDF 文档，无需跳转到外部应用。

### 文本查看器

支持 txt、log、json、xml 等常见文本格式的直接查看，方便开发者和普通用户快速预览文件内容。

### 图片查看器

全屏查看、双指缩放、左右滑动切换，流畅的图片浏览体验。

### 媒体播放器

内置音频和视频播放器，支持常见音视频格式的直接播放，无需跳转到第三方播放器。

### 基础清理

智能扫描缓存文件、临时文件、空文件夹、缩略图缓存、过期旧文件等，一键释放存储空间。

### 深度存储分析

可视化展示存储空间占用分布，智能识别大文件和重复文件，帮助你精准定位空间占用大户，高效管理存储。

### Material Design 3 UI

遵循 Material Design 3 设计规范，界面简洁现代，支持深色模式，视觉体验与系统风格统一协调。

## Pro 版 AI 功能

Pro 版解锁 8 项 AI 智能功能，基于端侧推理和云端协作，让文件管理从"手动操作"升级为"智能驱动"。

### 语义搜索

<div style="text-align: center; margin: 1rem 0;">
  <img src="/语义搜索.webp" alt="语义搜索" style="max-width: 280px; border-radius: 12px; box-shadow: 0 2px 12px rgba(0,0,0,0.1);" />

基于 MobileCLIP2 模型的端侧向量化引擎，用自然语言描述即可找到图片和视频。支持中英文查询，完全离线可用，文件内容绝不出端。

智能识别近重复图片和语义相似图片并自动分组，基于分辨率、清晰度、来源目录等维度提供智能推荐保留策略，帮助你高效去重。

AI 驱动的隐私文件扫描，自动识别身份证、银行卡、证件照、裸露、色情等敏感文件，配合密码保护的加密存储空间，为你的隐私文件提供双重保障。

## 技术亮点

- **原生 Kotlin + Jetpack Compose**：全原生构建，极致性能与流畅体验，告别 WebView 和跨平台框架的性能损耗。
- **VFS 统一文件系统抽象层**：统一的虚拟文件系统接口，屏蔽底层存储差异，为上层功能提供一致的文件访问能力。
- **ONNX Runtime Mobile 端侧 AI 推理**：基于 ONNX Runtime Mobile 的端侧推理引擎，MobileCLIP2 等 AI 模型在设备本地运行，文件内容不出端。
- **端云分层架构**：云端大模型负责意图理解与任务规划，端侧原生代码负责文件访问与安全执行，"大脑指挥肌肉"的协作模式。
- **BYOK 零加价模式**：用户自带 API Key 直连大语言模型，零加价、零中转，数据直达用户配置的 API 端点，不经过应用服务器。
